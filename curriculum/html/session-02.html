第2回：Cloudflare Hono入門

今日の目標: Honoプロジェクトを作成し、APIの基本（ルーティング、リクエスト処理）を学ぶ。

1. プロジェクト作成

npm create hono@latest my-app
cd my-app
npm install
npm run dev

2. コードの構造

import { Hono } from 'hono'
const app = new Hono()

app.get('/', (c) => {
  return c.text('Hello Hono!')
})

export default app

3. ルーティング

パスパラメータ

app.get('/user/:id', (c) => {
  const id = c.req.param('id')
  return c.json({ message: `ユーザーID ${id}` })
})

クエリパラメータ

app.get('/search', (c) => {
  const query = c.req.query('q')
  return c.json({ result: `「${query}」の検索結果` })
})

POSTリクエスト

app.post('/posts', async (c) => {
  const body = await c.req.json()
  return c.json({ receivedData: body }, 201)
})

4. エラーの読み方

エラーが出たら、最初の1行とファイル名:行番号を確認しましょう。

ハンズオン課題
