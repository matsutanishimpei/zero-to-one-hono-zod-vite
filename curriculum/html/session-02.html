第2回：HonoによるAPI開発の基本

今日の目標: APIのエンドポイントを作り、各種パラメータの受け取り方を理解する。

1. ルーティングの実装

パスパラメータ

app.get('/user/:id', (c) => {
  const id = c.req.param('id')
  return c.json({ message: `ユーザーID ${id} の詳細です。` })
})

クエリパラメータ

app.get('/search', (c) => {
  const query = c.req.query('q')
  return c.json({ result: `「${query}」の検索結果（仮）` })
})

2. POSTリクエストとJSON

データを送信する際は app.post を使用します。

app.post('/posts', async (c) => {
  const body = await c.req.json()
  return c.json({
    message: 'データを受け取りました！',
    receivedData: body
  }, 201)
})
