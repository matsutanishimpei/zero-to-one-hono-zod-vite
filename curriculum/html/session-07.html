第7回：実践アプリ構築とデプロイ

今日の目標: すべての技術を統合してアプリを完成させ、Cloudflare にデプロイして世界に公開する。

1. 最終課題：一言掲示板

テンプレートの TODO コメント部分を自分で実装しましょう。

  • 投稿フォーム + バリデーション (Zod)
  • メッセージの保存 + 一覧表示 (D1 + RPC)
  • 削除機能（余裕があれば）

2. DevTools の使い方

エラー原因対策
CORS errorcors() がないバックエンドに追加
404URL間違いパスを確認
400バリデーション失敗送信データを確認
500サーバーバグターミナルを確認

3. デプロイ

# バックエンド
npx wrangler login
npm run deploy

# フロントエンド
npm run build
npx wrangler pages deploy dist

おわりに

お疲れ様でした！公開URLをクラスメートに共有して、お互いのアプリを触ってみましょう。次は認証、画像保存、リアルタイム通信など、さらに高度な機能に挑戦してみてください。
