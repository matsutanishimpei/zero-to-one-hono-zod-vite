Cloudflare + Hono + Zod + Vite モダン開発カリキュラム

JavaScriptの基礎を触ったことがある学生を対象に、現代的なWeb開発スタックを90分×7回の授業で習得することを目的としたカリキュラムです。

授業スケジュール

回数 タイトル 主な内容
1 モダンJS復習とTypeScript入門 アロー関数、async/await、型注釈、型推論
2 Cloudflare Hono入門 プロジェクト作成、ルーティング、エラーの読み方
3 Zodによるバリデーション スキーマ定義、zValidator、z.infer
4 React入門 JSX、コンポーネント、useState、useEffect
5 Hono RPCによるフロント連携 型安全な通信、CORS設定
6 Cloudflare D1（データベース） Bindings、SQL基礎、CRUD操作
7 実践アプリ構築とデプロイ 統合、DevTools、Cloudflareへの公開
Ex モノレポ構成とコード共有 npm workspaces、パッケージ分割

推奨環境

  • Node.js: v20以上推奨
  • エディタ: VS Code (Prettier, ESLint)
  • アカウント: Cloudflare アカウント（無料）
