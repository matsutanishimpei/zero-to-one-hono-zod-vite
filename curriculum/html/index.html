Cloudflare + Hono + Zod + Vite モダン開発カリキュラム

このカリキュラムは、JavaScriptの基礎を触ったことがあるものの、ブランクがある学生を対象に、現代的なWeb開発スタックを90分×6回の授業で習得することを目的としています。

授業スケジュール

回数 タイトル 主な内容
1 モダンJS復習とCloudflare Hono入門 開発環境構築、HonoでのHello World
2 HonoによるAPI開発の基本 ルーティング、JSON通信、パラメータ取得
3 Zodによるデータバリデーション 型安全なデータ受け取り、エラーハンドリング
4 Vite + Hono RPCによるフロント連携 フロントエンド構築、型安全なAPI呼び出し
5 Cloudflare D1（データベース） サーバーレスDBへのデータ保存、SQL基礎
6 実践アプリ構築とデプロイ 知識の統合、Cloudflareへの公開
Ex モノレポ構成とコード共有 npm workspaces、型共有、パッケージ分割

推奨環境

  • Node.js: v20以上推奨
  • エディタ: VS Code
  • アカウント: Cloudflare アカウント（無料）
