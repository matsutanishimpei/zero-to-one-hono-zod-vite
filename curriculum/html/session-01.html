第1回：モダンJS復習とCloudflare Hono入門

今日の目標: Cloudflareエコシステムを理解し、Honoで「Hello World」を表示する。

1. JavaScript/TypeScriptの「これだけは」復習

現代的な開発で必須となる書き方を復習します。

// アロー関数
const greet = (name: string) => `Hello ${name}`;

// 非同期処理 (async/await)
const res = await fetch('https://api.example.com');

// 分割代入
const { name, age } = { name: 'Taro', age: 20 };

2. Cloudflare & Hono とは？

  • Cloudflare Workers: 世界中のエッジで動く高速なサーバーレス環境。
  • Hono: 日本発の世界標準フレームワーク。超軽量で爆速です。

3. ハンズオン：最初のプロジェクト

ターミナルで以下のコマンドを実行しましょう。

npm create hono@latest my-app
cd my-app
npm install
npm run dev

src/index.ts を編集してみましょう。

import { Hono } from 'hono'
const app = new Hono()

app.get('/', (c) => {
  return c.text('Hello Hono! 爆速でWebサーバーが立ち上がりました。')
})

export default app

まとめ

  • app.get('/', ...) でパスを設定。
  • c.text() でレスポンスを返却。
  • wrangler が開発サーバーを担当。
