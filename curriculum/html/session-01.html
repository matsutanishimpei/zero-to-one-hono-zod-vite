第1回：モダンJS復習とTypeScript入門

今日の目標: JavaScriptの現代的な書き方を復習し、TypeScriptの「型」の基本を理解する。

1. JavaScript 復習

アロー関数

const greet = (name) => 'Hello ' + name;

テンプレートリテラル

const name = 'Taro';
const message = `Hello ${name}!`; // "Hello Taro!"

async/await

const fetchData = async () => {
  const res = await fetch('https://api.example.com/data');
  const data = await res.json();
};

分割代入

const { name, age } = { name: 'Taro', age: 20 };

2. TypeScript とは？

JavaScript に「型」を追加した言語です。バグを書いた瞬間にエディタが教えてくれます。

// 型注釈：変数の後に ": 型名" を書く
const name: string = 'Taro';
const age: number = 20;
const isStudent: boolean = true;

// 型推論：明らかな場合は省略可能
const city = 'Tokyo'; // string と推論される

3. オブジェクトの型

type User = {
  name: string;
  age: number;
  email: string;
};

const user: User = {
  name: 'Taro',
  age: 20,
  email: 'taro@example.com'
};

まとめ

  • アロー関数、async/await、分割代入は必須知識。
  • TypeScript は「: string」のように型を書いてバグを防ぐ。
  • type でオブジェクトの形を定義できる。
