第5回：Hono RPCによるフロント連携

今日の目標: Hono RPC を使い、フロントエンドからバックエンドの関数を型安全に呼び出す。

1. 従来の問題

// URL文字列 → タイポしてもエディタが気付かない
const res = await fetch('/api/users')
const data = await res.json() // data は any 型

2. バックエンド側

import { Hono } from 'hono'
import { zValidator } from '@hono/zod-validator'
import { z } from 'zod'
import { cors } from 'hono/cors'

const app = new Hono()
app.use('*', cors())

const route = app.post(
  '/hello',
  zValidator('json', z.object({ name: z.string() })),
  (c) => {
    const { name } = c.req.valid('json')
    return c.json({ message: `こんにちは、${name}さん！` })
  }
)

export type AppType = typeof route
export default app

3. フロントエンド側

import { hc } from 'hono/client'
import type { AppType } from '../../backend/src/index'

const client = hc<AppType>('http://localhost:8787')

// 補完が効く！URLのタイポが不可能に
const res = await client.hello.$post({ json: { name: '学生' } })
const data = await res.json()
console.log(data.message)

ハンズオン

入力フォームから名前を送信し、レスポンスを React の State で画面に表示しましょう。client. の後に補完が効くことを確認してください。
