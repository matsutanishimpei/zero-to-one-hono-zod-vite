第4回：Vite + Hono RPCによるフロント連携

今日の目標: APIの型をフロントエンドと共有し、型安全な通信を実装する。

1. RPC機能のメリット

URLを文字列で書く必要がなくなり、補完が効くようになります。APIの変更にも即座に気づけます。

2. フロントエンドの準備

npm create vite@latest frontend -- --template react-ts
cd frontend
npm install hono

3. 連携コード

// Frontend (App.tsx)
import { hc } from 'hono/client'
import type { AppType } from '../../backend/src/index'

const client = hc('http://localhost:8787')

const fetchHello = async () => {
  const res = await client.hello.$post({ json: { name: '学生' } })
  const data = await res.json()
  console.log(data.message)
}

ハンズオン

サーバーから取得したデータを React の State に保存して、画面に表示させてみましょう。
