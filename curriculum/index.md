# Cloudflare + Hono + Zod + Vite モダン開発カリキュラム

このカリキュラムは、JavaScriptの基礎を触ったことがある学生を対象に、現代的なWeb開発スタック（TypeScript / Cloudflare / Hono / Zod / Vite / React）を90分×7回の授業で習得することを目的としています。

## 授業一覧

| 回数 | タイトル | 内容 |
| :--- | :--- | :--- |
| [第1回](session-01.md) | モダンJS復習とTypeScript入門 | アロー関数、async/await、型注釈、型推論 |
| [第2回](session-02.md) | Cloudflare Hono入門 | プロジェクト作成、ルーティング、エラーの読み方 |
| [第3回](session-03.md) | Zodによるバリデーション | スキーマ定義、zValidator、z.infer |
| [第4回](session-04.md) | React入門 | JSX、コンポーネント、useState、useEffect |
| [第5回](session-05.md) | Hono RPCによるフロント連携 | 型安全な通信、CORS設定 |
| [第6回](session-06.md) | Cloudflare D1（データベース） | Bindings、SQL基礎、CRUD操作 |
| [第7回](session-07.md) | 実践アプリ構築とデプロイ | 統合、DevTools、Cloudflareへの公開 |
| [特別回](session-monorepo.md) | モノレポ構成とコード共有 | npm workspaces、パッケージ分割 |

## 推奨環境
- Node.js (v20以上) または Bun
- VS Code (Prettier, ESLint 拡張推奨)
- Cloudflare アカウント
