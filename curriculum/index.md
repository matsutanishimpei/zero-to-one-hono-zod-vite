# Cloudflare + Hono + Zod + Vite モダン受講カリキュラム

このカリキュラムは、JavaScriptの基礎を触ったことがあるものの、ブランクがある学生を対象に、現代的なWeb開発スタック（Cloudflare / Hono / Zod / Vite）を90分×6回の授業で習得することを目的としています。

## 授業一覧

| 回数 | タイトル | 内容 |
| :--- | :--- | :--- |
| [第1回](session-01.md) | モダンJS復習とCloudflare Hono入門 | 開発環境構築、HonoでのHello World |
| [第2回](session-02.md) | HonoによるAPI開発の基本 | ルーティング、JSON通信、パラメータ取得 |
| [第3回](session-03.md) | Zodによるバリデーション | 型安全なデータ受け取り、エラーハンドリング |
| [第4回](session-04.md) | Vite + Hono RPCによるフロント連携 | フロントエンド構築、型安全なAPI呼び出し |
| [第5回](session-05.md) | Cloudflare D1（データベース） | サーバーレスDBへのデータ保存、SQL基礎 |
| [第6回](session-06.md) | 実践アプリ構築とデプロイ | 知識の統合、Cloudflareへの公開 |
| [特別回](session-monorepo.md) | モノレポ構成 | 複数プロジェクトの管理 |

## 推奨環境
- Node.js (v20以上) または Bun
- VS Code (Prettier, ESLint 拡張推奨)
- Cloudflare アカウント
