# 第6回：実践アプリ構築とデプロイ

## 1. 今日の目標
- これまでの知識（Hono, Zod, Vite, D1）を統合して一つのアプリを完成させる
- インターネット上に公開（クラウドへのデプロイ）する

## 2. 最終課題：一言掲示板アプリ
以下の機能を備えたアプリをチームまたは個人で構築しましょう。
- **入力フォーム**: 誰でも一言投稿できる。
- **バリデーション**: 空っぽの投稿や、長すぎる投稿（140文字以上など）を弾く。
- **一覧表示**: 投稿されたメッセージを新着順に表示する。
- **削除機能**: (余裕があれば) 特定のメッセージを削除できるようにする。

## 3. 世界への公開（デプロイ）
Cloudflareにログイン（`npx wrangler login`）し、デプロイコマンドを叩きます。

```bash
# バックエンドのデプロイ
npm run deploy

# フロントエンドのデプロイ (Cloudflare Pagesなど)
cd frontend
npm run build
npx wrangler pages deploy dist
```

発行されたURLを友達に送ってみましょう。自分の作ったものが世界中からアクセスできるようになります。

## 4. 振り返りと次へのステップ
- **Hono/Cloudflare**: インフラを気にせず、ロジックに集中できる楽しさ。
- **TypeScript**: 型があるからこそ安心してコードが書ける。
- **モダン開発**: 昔（サーバーを立ててPHPを置いて…）に比べて、今の開発がいかに高速で安全かを実感できたはずです。
